# OpenClaw Scripts and Tools

Automation scripts and tools for OpenClaw workflow management.

## Description

This repository contains useful scripts, tools, and utilities for managing and automating tasks with OpenClaw.

## Getting Started

### Prerequisites

- [OpenClaw](https://openclaw.ai) installed and configured
- [GitHub CLI (`gh`)](https://cli.github.com/) installed
- Node.js and npm

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/mgossman71/OpenClaw-Scripts_and_Tools.git
   cd OpenClaw-Scripts_and_Tools
   ```

2. Make scripts executable:
   ```bash
   chmod +x *.sh
   ```

## Available Scripts

### `gog-verify.sh` - Verify Google Workspace Access

Checks if Gmail and Calendar access is working correctly.

```bash
./gog-verify.sh
```

### `github-repo-create.sh` - Create OpenClaw-related Repositories

Quickly create new GitHub repositories for OpenClaw projects.

```bash
./github-repo-create.sh OpenClaw-Your-Project
```

## Tools

This directory may contain various tools for:

- **Configuration management**: Scripts to backup and restore OpenClaw configurations
- **Testing utilities**: Tools to verify OpenClaw integrations (Gmail, Calendar, etc.)
- **Automation**: Scripts for common OpenClaw operations
